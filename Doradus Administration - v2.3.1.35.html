Runtime Memory
The default JVM memory is probably for a Doradus Server doing anything important. Also, tests have shown that growing a Java processâ€™s memory can be very disruptive, therefore, we recommend setting both the minimum and maximum memory parameters to the same value. 4GB is a good value to start with. Example:
java -Xms4G -Xmx4G -cp "../lib/*:../config/*" com.dell.doradus.core.DoradusServer
