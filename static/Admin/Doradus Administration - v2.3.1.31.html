Delete Tenant
An existing tenant can be deleted via the following REST command:
DELETE /_tenants/{tenant}
Where {tenant} is the tenant name. No message body is provided in the command. The tenantâ€™s keyspace, all of its applications, and of its data are deleted. (Note that Cassandra creates a snapshot of a CF when it is deleted, so the data is still recoverable for a while.)
