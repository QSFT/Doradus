Doradus Query Language (DQL) : Clauses
Clauses
A DQL query is comprised of one or more Boolean expressions called clauses. Each clause examines a field that is related to perspective objects. The clause evaluates to true for a given object if the examined field matches the clauseâ€™s condition. Example clauses are shown below:
Name:Smith
FirstName=John
SendDate:[2001 TO "2005-06-31"]
NOT Name:Jo*
ALL(InternalRecipients).ANY(MessageAddress).Domain.Name = "dell.com"
Manager^(3).LastName:Wright
These examples show a variety of comparison types: qualified and unqualified fields, contains and equals, ranges, pattern matching, quantifiers, and transitive searches.
